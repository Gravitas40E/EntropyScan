# 🔐 EntropyScan

EntropyScan is a Python-based password strength analyzer that evaluates passwords using entropy calculation, regex validation, and breached password detection.

## 🚀 Features

* Password strength analysis
* Entropy calculation
* Crack-time estimation
* Breached password detection
* Colored terminal output

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Gravitas40E/EntropyScan.git
cd EntropyScan
```

Install dependency:

```bash
pip install colorama
```

Run the program:

```bash
python main.py
```

---

## 🔒 Breach Detection

Download `rockyou.txt`:

https://weakpass.com/wordlists/rockyou.txt

Place it inside the project folder:

```bash
EntropyScan/
│
├── main.py
├── rockyou.txt
└── README.md
```
